Musik beranda (landing page)
==========================

Letakkan file MP3 Anda di folder ini dengan nama PERSIS (huruf demi huruf):

  beranda-ambient.mp3
  ^^^^^^^^ ambient (ada huruf "t")

Salah ketik umum: beranda-ambien.mp3  <- TIDAK akan terbaca

Path lengkap di proyek:
  public/audio/beranda-ambient.mp3

URL di browser:
  /audio/beranda-ambient.mp3

Setelah file ada, refresh halaman beranda dan klik "Putar musik" di tengah atas.

Tips: gunakan file ringan (mis. 2–5 MB), format MP3, dan cocok untuk loop.
